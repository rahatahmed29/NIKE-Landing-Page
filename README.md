# Nike Landing Page

A modern, responsive Nike-inspired landing page built with **React**, **Vite**, and **Tailwind CSS**. The project showcases a clean, component-based architecture with a visually appealing user interface, responsive design, and reusable UI components.

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

Clone the repository:

```bash
git clone https://github.com/rahatahmed29/NIKE-Landing-Page.git
```

Navigate to the project:

```bash
cd NIKE-Landing-Page
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
